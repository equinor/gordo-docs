Reads a single iroc timeseries csv, and returns it as a pandas.DataFrame. The returned dataframe has timestamps as a DateTimeIndex, and upto one column per tag in tag_list, but excluding any tags which does not exist in the csv.

  • file_obj (str or path object or file-like object) – File object to read iroc timeseries data from
  • from_ts – Only keep timestamps later or equal than this
  • to_ts – Only keep timestamps earlier than this
  • tag_list – Only keep tags in this list.
Returns:

Dataframe with timestamps as a DateTimeIndex, and up to one column per tag in tag_list, but excluding any tags which does not exist in the csv.
